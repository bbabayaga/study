The Basic Language of the Web : Html

posted by laura jones on monday , june 21st 2027

all modern websites and web application are built using three fundamental technologies : html, css and javascript , these are the language of the web.

in this post , let's focus on html . we will learn what html is all about , and why you too should learn it.

what is HTML ?

HTML stand for HyperText Harkup Language. it's a markup language that web developers use to structure and describe the content of the webpage (not a programming language).

HTML consists of elements that describe different type of content: paragraphs , links, headings,images,video,etc. web browsers understand html and render HTML code as websites.

in HTML , each elements is made up of 3 parts :

  1. the opening tag
  2. the closing tag
  3. the actual element

you can learn more at MDN web Docs

why should you learn HTML?

there are countless reasons for learning the fundamental language of the web . here are 5 of them:

  • to be able to use the fundamental web dev language
  • to hand-craft beautiful website instead of relying on tools like word-press or wix
  • to build web applications
  • to impress friends
  • to have fun :D
